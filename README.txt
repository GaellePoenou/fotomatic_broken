# Fotomatic

## Description

Fotomatic est un site web axé sur la photographie avec un design élégant et moderne. 

## Technologies Utilisées

- HTML, CSS
- Google Fonts (Damion, Rubik, Roboto)
- Reset CSS

## Installation

1. **Clonez le dépôt** : `git clone https://github.com/votre-utilisateur/fotomatic.git`
2. **Ouvrez le fichier HTML** : `index.html`
